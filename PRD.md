# 轻量版 Markdown 预览与编辑器 - 产品需求文档（PRD）

> ⚠️ **技术栈变更说明**：PRD 最初版本（v0.x）规划使用 Flutter，**实际 v0.1.0 实现已切换为 Tauri 2 + React 19 + TypeScript**（详见 [README.md](README.md) 与 [DEV-DOC.md](DEV-DOC.md)）。本文档保留作为产品功能规划的参考，**1.3 节技术选型仅代表最初决策**，请以代码仓库为准。

## 1. 项目概述

### 1.1 产品定位
一款轻量、跨平台的 Markdown 预览与编辑软件，面向需要快速编辑、预览、管理 Markdown 文件的用户。第一版本即提供完整的核心编辑能力 + 常用进阶功能。

### 1.2 目标平台
| 优先级 | 平台 | 说明 |
|--------|------|------|
| P0 | Windows | 第一版本主要交付平台 |
| P1 | macOS / Linux | 第二版本覆盖桌面三端 |
| P2 | Android / iOS | 第三版本覆盖移动端 |

### 1.3 技术选型
- **开发框架**：Flutter + Dart
- **状态管理**：Riverpod
- **Markdown 渲染**：flutter_markdown
- **LaTeX 数学公式**：flutter_math_fork（渲染）+ markdown 扩展语法解析
- **Mermaid 图表**：WebView 或本地 JS 渲染（如 flutter_inappwebview / mermaid-js）
- **PDF 导出**：pdf 包 + 打印为 PDF（基础版）；自定义页眉页脚/页边距为会员功能
- **代码高亮**：highlight.dart
- **文件系统**：Dart io + path_provider
- **配置持久化**：shared_preferences / hive
- **桌面窗口管理**：window_manager

## 2. 功能需求

### 2.1 核心功能（免费版）

#### 2.1.1 文件操作
- [ ] 打开单个 Markdown 文件
- [ ] 新建空白 Markdown 文件
- [ ] 保存当前文件（Ctrl/Cmd + S）
- [ ] 另存为
- [ ] 关闭当前文件
- [ ] 关闭所有文件
- [ ] 未保存变更提示（关闭前/切换前）
- [ ] 最近打开文件列表（最近 20 个）
- [ ] 文件拖拽打开（桌面端）

#### 2.1.2 多标签页管理
- [ ] 每个打开的文件以独立标签页展示
- [ ] 标签页显示文件名 + 未保存状态标识（例如 `*`）
- [ ] 支持点击切换标签页
- [ ] 支持关闭单个标签页 / 关闭其他 / 关闭全部
- [ ] 支持标签页拖拽排序（桌面端）
- [ ] 双击标签栏空白处或快捷键新建文件
- [ ] 右键标签页菜单：关闭 / 关闭其他 / 复制路径 / 在文件夹中显示

#### 2.1.3 编辑与预览模式
- [ ] **编辑模式**：仅显示编辑器
- [ ] **预览模式**：仅显示 Markdown 渲染结果
- [ ] **编辑 + 预览模式**：左右分屏，实时同步滚动
- [ ] 模式切换通过工具栏按钮或快捷键完成
- [ ] 记忆每个文件上次使用的模式

#### 2.1.4 Markdown 编辑能力
- [ ] 基础语法编辑：标题、段落、列表、引用、代码块、链接、图片、表格、分隔线
- [ ] 任务列表（`- [ ]` / `- [x]`）
- [ ] 代码块语法高亮
- [ ] 编辑器显示行号
- [ ] 当前行高亮
- [ ] 可选：显示 / 隐藏行号
- [ ] 可选：自动换行切换

#### 2.1.5 Markdown 预览能力
- [ ] 实时渲染编辑内容
- [ ] 基础 HTML 标签支持
- [ ] GitHub Flavored Markdown 支持
- [ ] 代码块高亮渲染
- [ ] 表格渲染
- [ ] 任务列表渲染为可勾选复选框（预览模式下可交互）
- [ ] 图片本地路径自动解析
- [ ] **LaTeX 数学公式渲染**（行内公式 `$...$`、块级公式 `$$...$$`）
- [ ] **Mermaid 图表渲染**（流程图、时序图、类图、甘特图等）

#### 2.1.6 语法提示与快捷编辑
- [ ] Markdown 语法自动补全提示
- [ ] 输入 `- ` 后按 Tab 自动缩进为列表
- [ ] 输入 `1. ` 后回车自动补全下一个序号
- [ ] 输入代码块围栏（```）后回车自动补全结束围栏
- [ ] 快捷键快速插入：
  - 粗体（Ctrl/Cmd + B）
  - 斜体（Ctrl/Cmd + I）
  - 行内代码（Ctrl/Cmd + `）
  - 标题（Ctrl/Cmd + 1~6）
  - 插入链接（Ctrl/Cmd + K）
  - 插入代码块（Ctrl/Cmd + Shift + C）
  - 插入引用（Ctrl/Cmd + Shift + Q）

## 3. 非功能需求

### 3.1 性能
- 启动时间 ≤ 3 秒（Windows 普通配置）
- 打开 10MB 以内 Markdown 文件无明显卡顿
- 实时预览渲染延迟 ≤ 200ms

### 3.2 稳定性
- 崩溃或异常退出时尝试恢复未保存内容
- 文件保存操作具备原子性，避免写入过程中断导致文件损坏

### 3.3 兼容性
- 支持 `.md` / `.markdown` / `.mdx`（作为纯 Markdown 处理）文件
- 支持 Windows 10 / 11 第一版本交付

### 3.4 可扩展性
- 应用名称为 **MiniMD**（品牌名），第一版本仅交付 **Windows** 平台
- Markdown 扩展语法支持 **LaTeX 数学公式** 和 **Mermaid 图表**
- **PDF 自定义页眉/页脚/页边距** 为会员功能，第一版本预留接口和 UI
- **云端同步** 为会员功能，第一版本预留账号体系与同步接口
- 插件系统预留架构，不纳入第一版本

## 9. 会员功能规划

### 9.1 会员体系说明
- 第一版本采用 **免费 + 会员增值** 模式
- 会员功能在 UI 中可见，未激活时展示「会员专属」标识或引导入口

- [ ] 自定义页眉内容（支持文本、图片、页码变量）
- [ ] 自定义页脚内容（支持文本、图片、页码变量）
- [ ] 自定义页边距（上 / 下 / 左 / 右）

- [ ] 登录账号后开启云端同步

## 10. 安全与隐私

- 不主动联网（除会员同步功能外）
- 不收集用户文件内容
- 导出 HTML 时过滤危险脚本（预览沙箱化处理）
- 云端同步需用户主动登录并授权，传输过程加密

## 4. 界面设计需求

### 4.1 整体布局
```
+--------------------------------------------------+
|  菜单栏 / 工具栏                                  |
+----------+---------------------------------------+
|          |  标签栏                               |
|  文件树  +---------------------------------------+

- 保存状态
