# MiniMD v0.1.0 (Windows)

轻量级 Markdown 编辑与预览工具。

## 安装方式

运行 `MiniMD-v0.1.0.msi`，按向导完成安装。

安装完成后：
- 程序会安装到 `C:\Program Files\MiniMD\MiniMD\`
- `.md`、`.markdown`、`.mdx` 文件会自动关联到 MiniMD
- 双击本地 Markdown 文件即可启动 MiniMD 打开该文件

## 当前版本功能

- 新建 / 打开 / 保存 Markdown 文件
- 多标签页管理
- 三种编辑模式切换：编辑 / 预览 / 分屏
- Markdown 实时预览
- LaTeX 数学公式渲染
- Mermaid 图表渲染
- 明亮 / 暗黑主题（跟随系统）
- 自定义无边框窗口标题栏
- 基础快捷键（Ctrl+B 粗体、Ctrl+I 斜体、Ctrl+K 链接、Ctrl+S 保存）
- 双击本地 Markdown 文件直接打开

## 注意事项

- 第一版本仅支持 Windows 桌面端（x64）
- 安装需要管理员权限（用于写入 `C:\Program Files` 和注册表）
- Windows 10 部分版本可能需要额外安装 WebView2 Runtime
- 会员功能（PDF 高级导出、云端同步）已预留接口，尚未开通

## 卸载

通过 Windows「设置 → 应用 → 已安装的应用」找到 MiniMD 卸载即可。
